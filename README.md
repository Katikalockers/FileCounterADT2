# FileCounterADT2  
is a Java program that analyzes a text file and outputs various statistics including the total number of characters, the count of palindromes, tokens, emoticons, newlines, the longest token, and the average token length.  
  
Features  
Character Count: Counts the total number of characters in the file.  
Palindrome Count: Identifies and counts palindromes (words that are the same forwards and backwards).  
Token Count: Counts the total number of tokens (words) in the file.  
Emoticon Count: Counts the number of emoticons like :), :-D, etc.  
Newline Count: Counts the number of lines in the file.  
Longest Token Length: Finds the longest word (token) in the file.  
Average Token Length: Calculates the average length of the tokens in the file.
